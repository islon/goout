## 功能特点

- **按城市订阅** — 选择所在城市订阅日历，只接收你关心的活动，避免冗余提醒
- **多维智能筛选** — 按时间、类型、区县、地点、费用、亲子友好度筛选，无结果选项自动置灰
- **聚合多源数据** — 自动抓取图书馆、博物馆、科技馆、青少年宫、体育中心等官方渠道活动信息
- **一键日历订阅** — 支持 iOS / Android / Google Calendar / Outlook 订阅，活动更新自动同步
- **场馆指南** — 收录 3000+ 优质场馆，按城市/类型/区县筛选，即使没有活动也值得去
- **真实可溯源** — 每个活动均保留官方来源链接，拒绝第三方新闻网站链接
- **零广告 · 完全免费** — 无商业推广，不收集个人信息

## 快速开始

### 方式一：微信小程序（推荐移动端）

微信搜索「童行亲子日历」或扫码小程序码，享受更流畅的移动端体验。

### 方式二：按城市订阅到日历

1. 打开 [订阅页面](https://islon.github.io/goout/schedule.html)
2. 选择你的城市（深圳 / 广州 / 上海 / 北京 / 杭州 / 成都 / 重庆 / 南京 / 武汉 / 西安 / 珠海）
3. 选择平台（iOS / Android），点击「xx添加到日历」
4. 活动信息将自动同步到你的手机日历

### 方式三：浏览活动列表

直接访问 [活动日程](https://islon.github.io/goout/)，支持按城市、时间、类型、区县、地点、费用等多维度筛选。

### 方式四：场馆指南

访问 [场馆指南](https://islon.github.io/goout/venue-guide.html)，浏览 3000+ 优质场馆介绍，即使没有活动也值得去。

## 技术栈

### 前端

- **网页版**：纯前端静态页面（HTML + CSS + JavaScript），托管于 GitHub Pages
- **微信小程序**：微信小程序框架 + 云开发云函数（`miniprogram/` 目录）

### 数据处理

- **爬虫**：Python 定时抓取各官方渠道数据（`scripts/` 目录）
- **数据输出**：标准 ICS / RSS / JSON 格式（全量 + 按城市），兼容所有日历客户端
- **数据管理**：多级缓存架构（远程最新 → 本地缓存 → 打包兜底），三级数据加载（Tier1 近期活动 → Tier2 全量活动 → Tier3 历史活动）

### 云开发

- **云函数**：微信云开发云函数，用于合规检测、数据转发（`cloudfunctions/` 目录）
- **CI/CD**：GitHub Actions 定时自动更新数据
