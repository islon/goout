# 童行 — 城市亲子活动日历

> 最好的成长是一路相伴

**童行**是一个聚焦城市亲子活动的开源日历项目，致力于帮助家长快速发现身边的优质亲子活动——展览、讲座、科普、演出、体育赛事等一站式聚合，支持一键订阅到手机日历。

---

## 覆盖城市

| 城市 | 状态 | 数据源 |
|------|------|--------|
| **深圳** | 已上线 | 图书馆、博物馆、科技馆、青少年宫、体育中心、文化馆等 |
| 更多城市 | 规划中 | 欢迎贡献你所在城市的数据源 |

---

## 功能特点

- **聚合多源数据** — 自动抓取图书馆、博物馆、科技馆、青少年宫、体育中心等官方渠道活动信息
- **一键日历订阅** — 支持 iOS / Android / Google Calendar / Outlook 订阅，活动更新自动同步
- **多维筛选** — 按时间（今天/明天/本周/本月）、类型（展览/讲座/科普/演出/赛事）、区县、收费状态、亲子友好度筛选
- **真实可溯源** — 每个活动均保留原始来源链接，确保信息真实可靠
- **零广告 · 完全免费** — 无商业推广，不收集个人信息

## 快速开始

### 方式一：订阅到日历（推荐）

1. 打开 [订阅页面](https://islon.github.io/goout/)
2. 选择你的平台（iOS / Android）
3. 点击"添加到日历"或复制订阅链接
4. 活动信息将自动同步到你的手机日历

### 方式二：浏览活动列表

直接访问 [活动日程](https://islon.github.io/goout/schedule.html)，支持按时间、类型、区县、收费状态等多维度筛选。

## 技术栈

- 纯前端静态页面（HTML + CSS + JavaScript）
- Python 爬虫定时抓取各官方渠道数据
- 输出标准 ICS / RSS / JSON 格式，兼容所有日历客户端
- 托管于 GitHub Pages，零服务器成本
